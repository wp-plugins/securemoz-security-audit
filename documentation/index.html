B) Installation - top

Installation

  1. Download the ZIP package from SecureMoz.
  2. Open WordPress admin and go to Plugins -> Add New -> Upload. Browse for the ZIP file securemoz.zip on your computer and hit “Install Now”.
  3. Activate the plugin.

Post install tutorial

  1. Open WordPress on Left Menu click - "Waudit Security"
  2. First time use "Run Scan" to analyse your site
  3. Click on Tabs for instructions
  4. Second time use "ReRun Scan" to analyze your site check you score

Upgrade

  1. Deactivate and delete the plugin in your WP admin -> Plugins.
  2. Use the Add New -> Upload function to upload the new plugin ZIP file.
  3. Activate the plugin.

SecureMoz requires WordPress v3.1.

Configuring maximum script execution time

In order to minimize the impact on your server SecureMoz has a limit on the maximum number of seconds it's tests can run. If you have a very slow server there may be a need to increase this limit.
If you want to increase this limit open .htaccess and /add/change line which defines it.

// maximum number of seconds tests are allowed to run
php_value max_execution_time 300

D) FAQ - top

Will this plugin slow my site down?

Absolutely not. You may experience a slight slow down while tests are being run but that takes less than a minute.

Will it work on my theme?

Sure! SecureMoz works with all themes.

Will it work with my plugins?

Sure! SecureMoz works with all plugins.

What changes will SecureMoz make to my site?

SecureMoz will just give you the test results and suggest corrective measures with precise instruction. It will not make any changes to your site until you use the instuctions.

Is this plugin safe to use?

Of course, SecureMoz Will Upgrade Security On your websites .

Is this plugin legal to use?

Yes. It's your site you can do whatever you want with it. Running tests on other people's sites is illegal but SecureMoz can only perform tests on the WP it's installed on.
